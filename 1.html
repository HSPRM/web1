HTML

Hypertext Markup Language (HTML) is the standard markup language for creating web pages and web applications. Web browsers receive THML documents from a web server or from local storage and render them into multimedia web pages. HTML describes the structure of a web page semantically and originally included cues for the apperag.

HTML elements are the building blocks of THML pages. With HTML constructs, images and other objects, such as interactive forms, may be embedded into the rendered pager. It provides a means to create structured documents by denoting structural sematics for text links, quotes and other items. HTML elements are delineated by tags, written using angle brackets.
